Automated fiber coupling system

My most recent project was working on a system to automate fiber coupling for a relatively simple laser setup. This project was part of a differential equations course but didn't really use diffential equations; instead the key idea was to use a variant of the Nedler-Mead (simplex) method for finding peaks for maximum power transfer. Code available here.

Vortex droplet generator

Currently in progress! Working on creating a vortexer to create droplets for an AST microscope. Working with 3D modelling and embedded code for the actual vortexer, and a bit of chemistry for the droplet generation.

Crop residue detection

I was part of an aerospace design team that was trying to launch a satellite to measure crop residue. We tried a bunch of different statistical analysis methods and also did some machine learning; we presented our results at the 46th Canadian remote sensing symposium in Lethbridge. As per licensing agreements, I'm unable to share most of the code as of right now, but feel free to reach out if you have any questions about it!
