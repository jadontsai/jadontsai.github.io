Person tracking trashcan

This was a group project for an embedded systems course (MIE438). The premise of the project was to have a trashcan following you around using computer vision; we used an STM32 NUCLEO WB55RG board as our microcontroller, and used a phone to capture video that was then processed with OpenCV and sent to the microcontroller through BLE. Code available here. Presentation video here.

Automated home cooling system

This project is to address energy use in Ghana as part of an engineering design course (ESC204). It's pretty much a smart thermostat that controls windows to take advantage of convective cooling, we have a few sensors that feed into a PID control loop that selectively opens windows to a certain extent. A lot of the work in this project was focused on the engineering design process itself, which can be seen in our design portfolio. . We also did smoke testing! Control and sensor code available here.

Automated fiber coupling system

This project was for a system to automate fiber coupling for a relatively simple laser setup. This project was part of a differential equations course but didn't really use diffential equations; instead the key idea was to use a variant of the Nedler-Mead (simplex) method for finding peaks for maximum power transfer. Code available here.

Vortex droplet generator

Currently in progress! Pictures will be updated as the project progresses (they're currently of a very early design phase). Working on creating a vortexer to create droplets for an AST microscope. This involves 3D modelling and embedded code for the actual vortexer (we're trying to make it from scratch), and a bit of chemistry for the droplet generation.

Crop residue detection

I was part of an aerospace design team that was trying to launch a satellite to measure crop residue. We tried a bunch of different statistical analysis methods and also did some machine learning; we presented our results at the 46th Canadian remote sensing symposium in Lethbridge. As per licensing agreements, I'm unable to share most of the code as of right now, but feel free to reach out if you have any questions about it!
